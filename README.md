# TCP/UDP Клиенты и Сервер
Данный проект содержит реализации клиентов и сервера для протоколов TCP и UDP. Код написан на C и Python, адаптирован для macOS с использованием POSIX API. Сервер работает одновременно с TCP и UDP на разных портах, а клиенты могут подключаться и обмениваться сообщениями с сервером.

## Порты
- TCP сервер слушает порт **8080**
- UDP сервер слушает порт **8081**

## Компиляция
1. Компиляция серверного кода:
cd server
clang server.c -o server -pthread
2. Компиляция C-клиентов:
cd ../client
clang client_tcp.c -o client_tcp
clang client_udp.c -o client_udp

## Запуск
1. Запустите сервер:
cd server
./server
2. В другом терминале выполните клиентов по необходимости:
cd client
Python TCP клиент
python3 client_tcp.py
Python UDP клиент
python3 client_udp.py
C TCP клиент
./client_tcp
C UDP клиент
./client_udp

## Принцип работы
- Сервер создаёт отдельные потоки на TCP и UDP портах.
- TCP клиент устанавливает соединение, отправляет сообщение и получает ответ.
- UDP клиент отправляет датаграмму и ждёт ответа с таймаутом.
- Вся коммуникация происходит на localhost (127.0.0.1).

## Примечания
- Для корректной работы UDP клиента убедитесь, что он использует порт 8081 (тот же, что и UDP сервер).
- Сервер и клиенты поддерживают UTF-8 сообщения.
- Проект демонстрирует базовый пример сетевого взаимодействия с использованием POSIX API и Python socket.
