# VisualProg
Calculator and Music Player

Android-приложение "Calculator and Music Player": простой и функциональный медиаплеер с управлением треками, прогрессом воспроизведения и современной анимацией CircularSeekBar.

Особенности
Проигрывание аудиотреков с локального устройства

Красивая анимация прогресса трека через CircularSeekBar

Визуализация времени проигрывания в виде виниловой пластины

Рандомизация цвета центра пластинки при каждом треке

Современный минималистичный дизайн Material Design

Лист аудиотреков с возможностью выбора

Реализовано на Kotlin (Android SDK 33+, targetSdk 36)
